# Sprint 2 — Autenticação e Painel Admin

> Sprint dedicada à autenticação, proteção de rotas e primeiros fluxos internos
> do administrador.

***

## Objetivo

Entregar o fluxo de login, armazenamento do token, guards de rota e o núcleo do
painel administrativo para gestão de nós, documentos e usuários.

***

## Foco esperado

- `POST /auth/login`
- Store de autenticação com Zustand
- Interceptor Axios com Bearer token
- `ProtectedRoute` e `RoleGuard`
- Primeiras telas e hooks do painel admin
